MigrationLog

MigrationLog class final

A record of a single migration execution.

Each log entry captures the details of a migration step including the versions involved, timing, outcome, and any error information.

Example

final log = MigrationLog(
  timestamp: DateTime.now(),
  fromVersion: '1.0.0',
  toVersion: '1.1.0',
  outcome: MigrationOutcome.success,
  durationMs: 1250,
);

// Serialize for storage
final json = log.toMap();

// Deserialize
final restored = MigrationLog.fromMap(json);
