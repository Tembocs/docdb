MigrationStep

MigrationStep class final

Represents a single migration step in a migration path.

A migration step binds a MigrationStrategy to its execution context, including the direction (upgrade vs downgrade) and position in the overall migration sequence.

Example

final step = MigrationStep(
  strategy: AddEmailFieldMigration(),
  isUpgrade: true,
  sequenceNumber: 1,
);

// Execute the step
final migrated = step.isUpgrade
    ? await step.strategy.up(entities)
    : await step.strategy.down(entities);
