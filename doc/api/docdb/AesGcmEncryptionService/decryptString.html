Decrypts a base64-encoded string.

  • ciphertext: The base64-encoded IV + ciphertext.
  • aad: Optional Additional Authenticated Data.

Throws DecryptionException if decryption fails. Throws AuthenticationFailedException if authentication fails.

Implementation

@override
Future<String> decryptString(String ciphertext, {Uint8List? aad}) async {
  final combined = base64Decode(ciphertext);
  final decrypted = await decryptCombined(
    Uint8List.fromList(combined),
    aad: aad,
  );
  return utf8.decode(decrypted);
}
