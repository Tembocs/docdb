MigrationConfig

MigrationConfig class final

Configuration for the migration system.

Defines the current schema version, available migrations, and behavior options for the migration process.

Example

final config = MigrationConfig(
  currentVersion: '2.0.0',
  migrations: [
    AddEmailFieldMigration(),
    RenameFieldMigration(),
  ],
  autoMigrate: true,
  createBackupBeforeMigration: true,
);
